# CRUD Project

# About the project

First challenge of the [DevSuperior Bootcamp Spring React 3.0](https://devsuperior.com). 
##### Create a SpringBoot application with the five basic CRUD operations:

 - Pageable search
 - Single search
 - Insert
 - Update
 - Delete

# Technologies
 - Java 11
 - Spring Boot
 - JPA / Hibernate
 - H2
